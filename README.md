# gouploadr

flickr uploader

## Usage

gouploadr require access-token to upload photos.

```
$ gouploadr
```

You'll see flickr web page for authenticate. Make confirmation to the authentication, then type `Enter`.

## Installation

```
$ go get github.com/mattn/gouploadr
```
